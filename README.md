# 🌿 Renewable Energy Advisor (FIS + React PWA)

## 📌 Project Overview

My project addresses a critical challenge in energy decision-making by providing an intelligent **Renewable Energy Advisor system**. As a Field Engineer, I designed this solution to recommend optimal energy sources using **Fuzzy Logic (FIS)** combined with a modern **React Progressive Web App (PWA)** interface.

The system analyzes environmental and operational parameters to suggest the most suitable renewable energy option.

---

## 🚀 Features

* ⚡ Intelligent energy recommendation using Fuzzy Logic
* 📊 MATLAB-based simulation and analysis
* 🌐 React-based user-friendly web interface
* 📱 Progressive Web App (PWA) support
* ☁️ Deployed using GitHub Pages

## 🛠️ Installation & Setup

### Clone the repository

```
git clone https://github.com/NipunKavinda95/Renewable-Energy-Advisor.git
cd Renewable-Energy-Advisor
```

### Install dependencies

```
npm install
```

### Run locally

```
npm start
```

### Build project

```
npm run build
```

### Deploy to GitHub Pages

```
npm run deploy
```
